## Install laravel preset package

```bash

 
 
composer require pacificdev/laravel_9_preset
# Esegui comando preset
php artisan preset:ui bootstrap --auth
npm i

```

Update the vite config file

```bash
mv vite.config.js vite.config.cjs
npm run dev
```
